# SimpleEcommerceNew
This is a simple Ecommerce website 
🛒 ShopMart - Java Web E-Commerce Application
A lightweight, responsive E-commerce web application built using Java Servlets and JSP. The project features a modern dark-themed UI and handles the complete flow from user registration to order placement.

✨ Key Features
User Authentication: Secure Login, Registration, and OTP Verification system.

Product Management: Dynamic product grid with real-time quantity controls (Plus/Minus buttons).

Shopping Cart: Fully functional cart that calculates subtotals and the final grand total.

Checkout & Payment: A streamlined checkout process with a secure-looking payment interface.

Responsive UI: Premium dark-themed design using CSS Flexbox and Grid, optimized for various screen sizes.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Custom Grid/Flexbox), JSP

Backend: Java (Servlets)

Server: Apache Tomcat

Architecture: MVC (Model-View-Controller) pattern

🚀 How to Run
Clone the repository to your local machine.

Import the project into Eclipse or IntelliJ IDEA as a Dynamic Web Project.

Configure your Apache Tomcat server.

Run login.jsp or index.jsp on the server.
